# Comeat-Doc

## Contexte

### Comeat est une application mobile de réservation de repas chez des hôtes volontaires qui vendent leurs spécialités chez eux comme repas et le tout se fait en réservation par les convives (clients).
### L’application permet donc qu’un convive puisse réserver son repas selon son choix de spécialité et de choisir l’hôte du repas.
